# JSON to SQLite Parser Project

A simple yet powerful Python project that reads data from a JSON file and automatically stores it into an SQLite database. 

## 🚀 Features
* **JSON Parsing:** Reads complex data from `data.json`.
* **Database Integration:** Automatically creates and updates `company_data.db`.
* **Data Automation:** Converts raw JSON objects into structured SQL table rows.

## 📁 Project Structure
* `parser.py`: The main Python script that contains the logic.
* `data.json`: The source file containing the raw data.
* `company_data.db`: The SQLite database where data is saved.

## 🛠️ How to Run
1. Make sure you have Python installed.
2. Run the following command in your terminal:
   ```bash
   python parser.py
